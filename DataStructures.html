Data Structures - Javascript

Eight different data types: undefined, null, boolean, string, symbol, bigint, number, and object

Primitive Data Structures and Data Types

  • boolean, null, number, string

    • Non Primitive Data Structures

  • linear data structures, static data structures, dynamic data structures

    • Arrays

  • Fixed Size. An array has a number of fixed cells decided on its creation.
  • Each cell has a numeric index used to select its data.
  • Can be used to create more complex data structures.
  • Inefficient to sort.

    • Queues

  • Sequencial structure that processes elements in the order they were entered.
  • FIFO version of Stacks
  • Helpful as a buffer for requests, storing each request in the order it was received until it can be processed
  • Effective as a buffer when receiving frequent data
  • Convenient way to store order-sensitive data such as stored voicemails

    • Linked List

  • linked lists use a referencing system: elements are stored in nodes that contain a pointer to the next node, repeating until all nodes are linked
  • This system allows efficient insertion and removal of items without the need for reorganization.
  • Efficient insertion and removal of new elements
  • Inefficient to retrieve a specific element
